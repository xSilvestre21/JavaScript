# Estudos de JavaScript

Repositório criado para organizar meus estudos, exercícios e projetos práticos em JavaScript.

## Objetivo 

Registrar minha evolução na linguagem JavaScript, praticando desde os fundamentos até conceitos mais avançados usados no desenvolvimento web. 

## Conteúdos estudados

- Variáveis e tipos de dados
- Operadores
- Condicionais
- Laços de repetição
- Funções
- Arrays
- Objetos
- Manipulação do DOM
- Eventos
- Promises
- Async/Await
- Consumo de APIs
- JavaScript moderno ES6+

## Como executar os arquivos

```git clone <url-do-repositorio>
cd nome-do-projeto
npm install
```
Para rodar arquivos JavaScript no terminal, use:

```bash
node nome-do-arquivo.js
```

Exemplo:

```bash
node variaveis.js
```

Para projetos com HTML, abra o arquivo:

```text
index.html
```

no navegador.
